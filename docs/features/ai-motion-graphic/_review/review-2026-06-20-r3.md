## Root cause (AC-04 reload-persistence)

The snapshot lives in `motion_graphic_block_snapshots` (FK `storyboard_block_media.motion_graphic_snapshot_id`), but:
- the storyboard read path (`storyboard.repository.ts` `findBlocksByDraftId` / `findBlocksByDraftIdForUpdate`) never JOINed the snapshot table — GET returned motion_graphic media with no code/duration/geometry;
- the frontend `BlockMediaItem` / `ModalMediaItem` types excluded `motion_graphic`, so `SceneModal.mediaSection` rendered it through the audio placeholder with an empty badge;
- the save/replace path + PUT schema would have dropped/rejected motion_graphic rows once the type was widened (delete+reinsert without the snapshot FK; `blockMediaItemSchema` required a non-null uuid `fileId` and an enum without `motion_graphic`).

## Fix (cohesive, backend + frontend, TDD)

Backend (apps/api):
- `repositories/storyboard.repository.types.ts` — widened `BlockMediaItem.mediaType` / `BlockMediaRow.media_type` to include `motion_graphic`, made `fileId` nullable, added a `motionGraphic` snapshot shape + joined `mg_*` columns + shared `mapBlockMediaRow`.
- `repositories/storyboard.repository.ts` — both read functions LEFT JOIN `motion_graphic_block_snapshots` (+ `motion_graphics` for title) via a shared select/group helper; `replaceStoryboard` INSERT now writes `motion_graphic_snapshot_id` (round-trip safe).
- `controllers/storyboard.controller.schemas.ts` + `storyboard.controller.ts` — `blockMediaItemSchema` accepts `motion_graphic`, nullable `fileId`, and a snapshot id (fileId-vs-snapshot exclusivity refine); controller normalizes the snapshot id into the repo shape.

Frontend (apps/web-editor):
- `features/storyboard/types.ts`, `components/SceneModal.types.ts` — widened block-media + modal-media kinds (`AssetKind | 'motion_graphic'`) + snapshot shape.
- `components/SceneModal.tsx`, `SceneModal.mediaSection.tsx`, `SceneModal.mediaPreview.tsx`, `hooks/useSceneModal.ts`, `StoryboardPage.bulkStreamUrls.ts`, `LibraryPanel.tsx` — pass the snapshot through autosave round-trip; render `MotionGraphicPlayer` (`data-testid="persisted-motion-graphic-preview"`) + an amber "MOTION GRAPHIC" badge for persisted motion_graphic items instead of the audio placeholder; templates filter out motion_graphic.
